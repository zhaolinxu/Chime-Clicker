Planned Features

  • Runes - Earn points after reseting game. Use points to buy runes. Runes grant bonus stats for future games.
  • Achievements - Earn achievements for completing specific tasks.
  • Price on hover of +10, +100, +1000 - Show total cost required for bulk purchases.
  • Simulate time lost in background tab - Game won't run when in the background. Simulate lost time when coming back to the game.

Known Bugs

When reporting bugs, include [Bug] in the subject line/title of any email/reddit post along with any information you think might be relevant, most notably which browser you are using (most testing has been done in Chrome, so formatting/layout issues across other browsers and on mobile are anticipated). Screenshots and steps to reproduce will go a long way in helping resolve any issues you guys encounter. Thanks in advance!

  • [All] Offscreen tooltips - With narrow window, tooltips for offscreen elements are misaligned.
  • [Fixed] Item cost on refresh - Item costs are wrong following refresh.
